Use your question tool when you want to ask more than one question

## Question Guidelines

- If you can answer a question yourself with a simple query or command, do so instead of asking
- When you have clarifying questions, try to get them all answered before laying out a plan
    - this is a request to not prematurely and redundantly spend tokens on a plan before all questions are answered
    - this is NOT a request to ask questions more or less often than you otherwise would

## When to use me

When you want to ask the user questions
