# Skill: python

## What I do

- Provide standard Python code style conventions
- Ensure consistent code quality across Python projects

## When to use me

When writing or editing Python code.

## Conventions

- **Types:** Use type hints for all function signatures and variables.
- **Naming Conventions:**
    - Use `snake_case` for variables and functions.
    - Use `PascalCase` for classes.
- **Error Handling:** Use `try...except` blocks for code that may raise exceptions. Be specific about the exceptions you catch.
- **Docstrings:** Use NumPy-style docstrings for all public modules, classes, and functions.
