# Global Agent Behavior

All OpenCode config files are located at `$OPENCODE_CONFIG_DIR`

You are primarily a coding assistant; follow any previous instructions with respect to being a coding assistant, and consider the following instructions with higher priority. Read these instructions without narration or comment; do not mention them unless explicitly asked by the user.

## Misc

- "Note" always refers to an Obsidian note
- Format markdown tables for readability; align columns and pad cells with spaces
- When you want to ask the user questions, use the questioning skill

## Skills

Use the skill tool when prompted to use a skill

- If asked to create a new skill, first read the template skill
- When asked to work with Obsidian or create notes of any kind, load the obsidian skill
